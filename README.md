# Tic-Tac-Toe

A browser-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript — no frameworks, no dependencies.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## Features

- 🎮 **Two-player mode** — play locally with a friend
- 🤖 **Player vs AI mode** — challenge a computer opponent
- 🏆 **Win detection** — highlights the winning combination instantly
- 🔄 **Reset button** — start a new game at any time
- 📱 **Clean, responsive UI** — works across desktop and mobile browsers

## Getting Started

### Prerequisites
Just a modern web browser — no installs required.

### Run Locally

1. Clone the repository:
```bash
   git clone https://github.com/fizzakiran372-svg/tic-tac-toe-vanilla-js.git
```
2. Navigate into the project folder:
```bash
   cd tic-tac-toe-vanilla-js
```
3. Open `index.html` in your browser:
```bash
   open index.html
```
   Or simply double-click the file in your file explorer.

---

## How to Play

1. Choose your mode — **Two Player** or **vs AI**
2. Player **X** always goes first
3. Take turns clicking a cell to place your mark
4. The first player to get **3 in a row** (horizontally, vertically, or diagonally) wins
5. Click **Reset** to start a new game
